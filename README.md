🌍 Tourist Companion – India's Virtual Tourist Guide

Tourist Companion is an AI-powered web application that serves as India's Virtual Tourist Guide, helping travelers plan smarter and more enjoyable trips across the country. The platform provides personalized trip planning, destination information, weather-based travel recommendations, interactive map navigation, and an AI chatbot for instant travel assistance. It also highlights local festivals, regional cuisine, and a community forum where travelers can share experiences and recommendations.

✨ Features

🤖 AI-powered travel assistant
🗺️ Smart trip planning
📍 Interactive map navigation
🌦️ Weather-based travel recommendations
🏛️ Detailed destination information
🎉 Festival and cultural information
🍛 Local food recommendations
👥 Community forum for travelers
📱 Responsive and user-friendly interface
